🎨 自定义 CSS
📋 可用的 CSS 类名：
.messages-container — 消息区域背景色
.message-ai .message-bubble — AI气泡
.message-user .message-bubble — 用户气泡
.message-ai .message-text — AI文字颜色
.message-user .message-text — 用户文字颜色
.message-time — 时间戳文字
.avatar-placeholder — 头像占位（无图时）
💡 示例（暗夜红风格）：
.messages-container { background: #1a1a2e; }
.message-ai .message-bubble {
  background: #16213e;
  border-radius: 4px 18px 18px 18px;
  border: 1px solid #0f3460;
}
.message-user .message-bubble {
  background: #e94560;
  border-radius: 18px 4px 18px 18px;
}
.message-ai .message-text { color: #e0e0e0; }
.message-user .message-text { color: #fff; }
.message-time { color: rgba(255,255,255,0.3); }
